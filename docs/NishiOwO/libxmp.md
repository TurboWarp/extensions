# Libxmp
This extension allows you to play tracker modules files on TurboWarp.

## Blocks

```scratch
play URL [https://...] with repeat count (-1) :: #2050a0
```
Plays the tracker module located at URL with repeat count. If you specify repeat count -1, it will repeat forever until you stop it.

---

```scratch
stop all playing :: #2050a0
```
Stops all playing tracker modules.
