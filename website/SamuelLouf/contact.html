Contact extension documentation

About this extension

This extension allows you to send e-mails, call people, and use Skype.

Blocks

Skype Blocks


Calls an user on Skype.

The string area is where you put the person you want to call's username.


Starts a chat discussion with an user on Skype.

The string area is where you put the person you want to chat with's username.


Shows the information page of an user on Skype.

The string area is where you put the person you want to see the information page's username.


Sends a file to an user on Skype.

The string area is where you put the person you want to send a file to's username.


Add an user to contacts on Skype.

The string area is where you put the person you want to add to your contacts's username.


Send a voice mail to an user on Skype.

The string area is where you put the person you want to send a voice mail to's username.


Phone and mail Blocks


Send a e-mail.

The first string area is where you can put the name of the e-mail.
The second string area is where you put the adress of the e-mail's destinator.
The third string area is where you can put the e-mail's content.



This block is used to adress someone personally.

The first string area is where you put the name of the person.
The first string area is where you put the e-mail adress of the person.




This block can be used to send an e-mail to multiple persons.

The string areas are where you put the destinators of the e-mail.

Tip: You can put the "person named..." blocks in the string areas.




Pass a call.

The string area is where you put the phone number to call.
